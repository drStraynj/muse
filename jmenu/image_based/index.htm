QuickMenu Pro V4.0 (Image Based)
Image Based Version Information - The QuickMenu Pro Image Version allows for the most flexibility with both the look and placement of the main menu items.  Each sub menu created with the menu system may be attached to any existing image tag within any HTML page.

Attachment is accomplished by pasting a simple statement within each image tag which is to act as a main menu item for QuickMenu Pro. Additional rollover effects for the main menu images may be optionally defined within QuickMenu Pro's data file.  The separate text based data file contains all the settings for the sub menus and main menu rollover images using simple plain English parameter settings. The integrated rollover image functionality allows this version to also act as a highly efficient rollover script for other image based buttons on the page which do not require sub menus.

This version of QuickMenu Pro is perfect for sites requiring unique main menu items or complex rollover effects.  Any type of image which is capable of displaying on an HTML page using the <img.. tag may be used.   If you do not wish to create images for your main menu items, the text based version will create main menu items for you. (includes an optional main menu icon images feature and is included in this download)



 Getting Started - QuickMenu Pro includes documentation covering all customizable parameters, and a documented sample implementation for fast customization. OpenCube recommends customizing your menu around the documented sample. This sample provides a quick reference to many of the customizable options the menu has to offer, and in most cases speeds up the design process.

Documented Sample Implementation

To access and modify the documented sample implementations settings, open the JavaScript file labeled sample_settings.js (located in the 'image_based' folder) in a simple text editor (Notepad or similar is recommended). To customize your menu simply modify and add values to suit your needs, view your changes by opening sample.htm (located in the 'image_based' folder) in a browser. For more information on the parameters and getting started refer to the documentation. The source of the sample file (sample.htm) also includes information on how to place the effect within your web page.
