QuickMenu Pro Information
QuickMenu Pro (text based version) is designed to be quick loading, quick running, and quick to implement. The menu contains more than 100 simple to use customizable parameter options. While capable of adapting to the design of any web site, QuickMenu retains a small footprint of 15K or less depending on the browser and platform used to view the effect.

QuickMenu Pro works to 100% capability with...

         IE4, IE5.x, IE6 (Win 95, 98, ME, 2000, NT, XP)
         IE4, IE5.x, & up (Mac)
         IE4 & Up (Other Platforms)
         NS4.x (All Platforms)
         NS5/6.x (All Platforms)
         NS7 (All Platforms)
         Opera 5,6,7  (All Platforms)
         Mozilla 0.6 & Up (All Platforms)
         Konqueror 2.2 & Up (Linux)
         Espial Escape 4.x & Up (All Platforms)
         Ice Browser 5.x & Up (All Platforms)
         Safari 1.0 (Mac only browser)

QuickMenu Pro includes a simple to use separate data file containing plain English parameter settings. By using a text editor to modify the parameters all aspects of the menus look and functionality may be customized to blend with any web site layout. The following table outlines some of QuickMenu's capabilities and available features.

QuickMenu Pro Features
Conditional file loading for different browser / platform scenarios for reduced code size.
Separate data file support with simple plain English parameter settings and layout.
Create vertical and horizontal main menu bars
Customizable mouse-off delay timing, with enhanced mouse movement detection for simplified menu selection..
Customizable border widths & colors, with transparency option.
Customizable divider heights & colors, with transparency option.
Customizable background colors and foreground highlight colors upon mouse over
Fully customizable font characteristics and colors.
Specify different border settings, divider settings, background colors, text colors, and more for specific sub menus.
Customizable left, right, top, and bottom margin settings.
Customizable left, center, or right text alignment.
Define unlimited relative positioned icon images and associate them with any number of sub menu items or main menu items.
Relative positioned icons automatically adjust position according to text alignment.
Define unlimited absolute positioned icon images and associate them with any number of sub menu items.
Specify absolute positioned icon image coordinates relative to the right corner or left corner of each sub menu item according to text alignment.
Customizable relative and absolute positioned icon rollover images.
Optional status bar display of sub menu item URL's.
Customizable status bar text for main menu and sub menu items.
Optionally Customizable main menu rollover images.
Div lay-over technique employed for rollover images in Internet Explorer - this technique works around the delayed image swap issue and allows for the use of animated GIF rollovers.
Customize sub menu items using plain text or HTML.
Optional customizable rollover text or HTML for each sub menu item.
Main menu and sub menu URL links with custom targeting options.
Define custom code or functions and attach them to any of the 4 exposed QuickMenu events.
Fully customizable sub menu transition effects for IE5.5 & up.
Fully customizable sub menu item rollover transition effects for IE5.5 & up.
Choose from 10 different transition effects and customize different aspects of each effect including duration times.
Customizable sub menu opacity for Internet Explorer.
Customizable sub menu drop shadows for IE5.5 & up.
Define enhanced transitions and sub menu looks for Internet Explorer while keeping the complete sub menu look and functionality with other browsers.
Support for the Opera 7 zoom feature, this makes the menu compatible with wireless devices and Pocket PC's incorporating the Opera 7 browser.
Full support for Konqueror, Opera, and other spoofing browsers, regardless of their identification settings.
